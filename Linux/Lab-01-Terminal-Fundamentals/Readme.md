# Lab 01 - Terminal Fundamentals

## Objective

The objective of this lab was to become familiar with the Linux terminal, basic navigation commands, and the Linux filesystem hierarchy.

---

## Lab Environment

* VMware Workstation Pro
* Kali Linux
* NAT Network Configuration
* Lenovo Laptop

---

## Commands Practiced

### pwd

Displays the current working directory.

### ls

Lists files and directories.

### ls -la

Lists files including hidden files and displays detailed information.

### cd

Changes the current directory.

---

## Procedures Performed

1. Opened the Kali Linux terminal.
2. Identified the current working directory using the `pwd` command.
3. Listed directory contents using `ls`.
4. Displayed hidden files and detailed information using `ls -la`.
5. Navigated between directories using the `cd` command.
6. Explored the Linux filesystem hierarchy.
7. Practiced relative and absolute path navigation.

---

## Key Findings

* Linux filesystems begin at the root directory (`/`).
* User home directories are stored under `/home`.
* Configuration files are commonly located in `/etc`.
* System logs are commonly located in `/var/log`.
* Relative paths depend on the current location.
* Absolute paths always start from the root directory.
