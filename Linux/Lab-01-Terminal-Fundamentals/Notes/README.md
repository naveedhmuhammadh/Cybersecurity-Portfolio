# Lab 01 Notes

## Terminal

The terminal is a command-line interface used to interact directly with the Linux operating system.

## pwd

Print Working Directory.

Displays the current location in the filesystem.

Example:

pwd

Output:

/home/kali

## ls

Lists files and directories within the current location.

## ls -la

Lists all files, including hidden files, with detailed information.

## cd

Changes the current directory.

Examples:

cd Documents

cd /home/kali

cd ~

## Relative Path

A path that starts from the current location.

Example:

cd Documents

## Absolute Path

A path that starts from the root directory (/).

Example:

cd /home/kali/Documents

## Home Directory

The symbol ~ represents the current user's home directory.

For user kali:

~

equals

/home/kali

## Important Directories

/ = Root directory

/home = User directories

/etc = Configuration files

/var/log = System logs

## Personal Learning Notes

The most important concept learned during this lab was understanding the difference between relative paths and absolute paths and how Linux organizes files in a hierarchical structure.
