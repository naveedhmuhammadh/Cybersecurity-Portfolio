# Lab 02 Notes

## mkdir

### Meaning

Make Directory

### Purpose

Creates a new directory.

### Example

mkdir TestLab

---

## touch

### Purpose

Creates a new empty file.

### Example

touch file1.txt

---

## cp

### Meaning

Copy

### Purpose

Creates a duplicate of a file.

### Example

cp file1.txt file2.txt

---

## mv

### Meaning

Move

### Purpose

Moves files or directories.

Also used for renaming.

### Example

mv file2.txt renamed-file.txt

---

## rm

### Meaning

Remove

### Purpose

Deletes files.

### Example

rm file1.txt

---

## rmdir

### Meaning

Remove Directory

### Purpose

Deletes an empty directory.

### Example

rmdir TestLab

---

## Output Redirection

### >

Writes output to a file.

If the file already contains data, it will be overwritten.

### Example

echo "Hello" > notes.txt

---

## Append Redirection

### >>

Appends output to the end of a file.

Existing content is preserved.

### Example

echo "Second Line" >> notes.txt

---

## Relative Path

Navigates based on the current location.

Example:

cd Documents

---

## Absolute Path

Navigates directly from the root directory.

Example:

cd /home/hex/Documents

---
